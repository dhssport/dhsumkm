# DHA Studio UMKM

Landing page untuk layanan solusi bisnis UMKM, menampilkan layanan konsultasi, pemasaran digital, dan pengembangan bisnis.

## Fitur
- Desain responsif dan modern
- Bagian layanan, portfolio, dan kontak
- Integrasi WhatsApp untuk konsultasi

## Cara Menggunakan
Cukup buka `index.html` di browser Anda untuk melihat website.

## Hosting di GitHub Pages
1. Push repository ini ke GitHub.
2. Buka Settings > Pages.
3. Pilih branch `main` (atau `master`) dan folder root `/`.
4. Website akan live di `https://username.github.io/repo-name`.
